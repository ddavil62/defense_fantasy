# 경제 시스템

Gold (인게임 화폐) + Diamond (메타 화폐) + 메타 업그레이드 + 컬렉션 모드.

## Gold 시스템

- 초기: 250G (유틸리티 Gold Boost 적용 시 최대 350G)
- 적 처치 시 타입별 보상 (Normal 5G, Fast 3G, Tank 15G, Boss 100G, Swarm 2G, Splitter 12G, Armored 20G)
- 웨이브 클리어 보너스: `round * 6 + floor(currentHP / maxHP * 25)`
- 소비: 타워 설치, 업그레이드 (Lv.2, Lv.3)
- 판매 환급: 총 투자 비용의 60%

## 기지 HP 시스템

- 초기 HP: 20 (유틸리티 Base HP+ 적용 시 최대 30)
- 회복 없음
- HP <= 5 시 빨간 깜빡임
- 적 기지 통과 시 빨간 플래시 이펙트
- HP 0 → 게임 오버

## Diamond 메타 화폐 (Phase 3)

### 획득 공식

```
floor(round / 5) * 2 + floor(round / 10) * 3
```

| 도달 라운드 | Diamond 획득 |
|---|---|
| R5 | 2D |
| R10 | 7D |
| R15 | 6D |
| R20 | 11D |
| R30 | 18D |

- 게임 오버 시 획득, 영구 축적
- 게임 오버 화면에 획득량/총량 표시

### 소비처

- 타워 메타 업그레이드 (컬렉션 모드)
- 유틸리티 업그레이드 (컬렉션 모드)
- 드래곤 타워 해금 (50 Diamond)

## 컬렉션 모드 (Phase 3)

메뉴 화면의 COLLECTION 버튼으로 진입.

### 타워 카드 그리드

- 4-4-2 레이아웃 (10종), 76x90px 카드
- 타워 색상/도형 표시
- 드래곤: 잠금 카드 (자물쇠), 클릭 시 50D 해금 팝업

### 타워 메타 업그레이드 트리

- 타워당 3티어 x A/B 분기 (총 60개 옵션)
- 영구 스탯 보너스 (damage, range, cooldown 등 multiply/add)
- 인게임 타워 배치/업그레이드 시 자동 적용
- Diamond 소모

### 유틸리티 업그레이드 (3종, 각 3티어)

| 업그레이드 | 효과 |
|---|---|
| Base HP+ | 20 → 22 / 25 / 30 |
| Gold Boost | 250 → 275 / 300 / 350G |
| Wave Bonus x | 1.0 → 1.1 / 1.2 / 1.3 |

### 세이브 데이터 마이그레이션

- Phase 2 → Phase 3 호환
- `migrateSaveData()` 함수로 누락 필드 자동 추가
