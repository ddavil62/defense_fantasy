# 적 시스템

6종 일반 적 + 방어형 보스, 디버프 시스템, 면역 시스템.

## 몬스터 컨셉

각 적 유형에 판타지 몬스터 컨셉을 매핑하여 5개 월드별 바이옴 스킨 변형을 지원한다.

| 유형 | 컨셉 | 바이옴별 이름 |
|------|------|-------------|
| normal | 고블린 (Goblin) | 숲 고블린, 사막 고블린, 설원 고블린, 화산 고블린, 그림자 고블린 |
| fast | 늑대 (Warg) | 숲늑대, 자칼, 빙랑, 지옥견, 그림자 늑대 |
| tank | 오우거 (Ogre) | 숲 오우거, 사막 거인, 예티, 용암 트롤, 어둠 거인 |
| swarm | 미니 슬라임 (Mini Slime) | 초록 미니, 모래 미니, 얼음 미니, 불 미니, 그림자 미니 |
| splitter | 슬라임 (Slime) | 녹색 슬라임, 모래 슬라임, 얼음 슬라임, 용암 슬라임, 그림자 슬라임 |

| boss_armored | 골렘 (Golem) | 나무 골렘, 사암 골렘, 얼음 골렘, 용암 골렘, 흑요석 골렘 |

### 스프라이트 시스템

- 스프라이트 파일: `public/assets/enemy/{worldId}_{spriteBase}.png`
- `getEnemySpriteKey(enemyType, worldId)` — 스프라이트 텍스처 키 생성
- 스프라이트 미존재 시 기존 도형으로 자동 폴백
- 아트 프롬프트: `docs/monster-art-guide.md` 참조

### 골렘 (방어형 보스)

- R20: HP=1500, 이동속도=25px/s, Gold=150G, 기지 피해=15, resistance=0.3
- R20+ 보스 라운드: 방어형 보스(boss_armored) 사용

### 보스 강화 (Phase 5)

- HP 2.5배, 속도 1.2배, Gold 1.5배
- 라운드별 저항 증가 (R30: 0.35, R50: 0.45, 캡 0.55)
- 도형 크기 확대 (boss: 26px, boss_armored: 30px)
- HP 바 강화 (두껍고 넓게, 금색 테두리)

## 디버프 시스템

| 디버프 | 효과 | 관련 타워 |
|---|---|---|
| 슬로우 (Slow) | 이동속도 감소 | 얼음, 번개(B), 마법사(B) |
| 화상 (Burn) | 지속 피해 DoT | 불꽃 |
| 독 (Poison) | 지속 피해 DoT (스택 가능, 2중첩) | 독안개 |
| 방어력 감소 (Armor Reduction) | resistance 증가 (취약) | 독안개(A) |
| 밀치기 (Pushback) | 경로 역방향 이동 | 바람 |
| 완전 정지 (Freeze) | 이동속도 0 (슬로우 100%) | 얼음(A), 바위(B) |

## 면역 시스템

`ENEMY_STATS`의 `immunities` 배열로 적 타입별 상태이상 면역을 선언한다.

- `boss`: `immunities: ['pushback']`
- `boss_armored`: `immunities: ['pushback']`
- `isImmune(effectType)` 게이트 메서드로 디버프 적용 전 체크
- 확장 가능: 새 적 타입에 면역 추가 시 config만 수정

## 이동 & 시각

- 웨이포인트 기반 경로 이동
- 바이옴별 몬스터 스프라이트 (스프라이트 미존재 시 도형 폴백)
- 스프라이트 모드: 디버프 틴트 (슬로우=파랑, 화상=빨강, 독=초록) + 반투명 오버레이
- 도형 모드: 기존 색상 도형 + 반투명 디버프 오버레이
- HP 바 표시 (녹 > 노 > 빨 색상 변화)
- 사망 파티클 (일반 8개, 보스 12개 + 대형 링)
- 분열형: 사망 시 현재 위치에서 미니 슬라임 3마리 소환 (동일 바이옴 스프라이트)
